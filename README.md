# Mgame
Mgame - «Who Wants to Be a Millionaire?» game

## Instructions

Set correct node version:

```shell
nvm use
```

Install npm packages:

```shell
npm i
```

Start the game:

```shell
npm start
```

Open url to play game:

http://localhost:3000

Start Storybook:

```shell
npm run storybook
```

Open url to review storybook:

http://localhost:6006
